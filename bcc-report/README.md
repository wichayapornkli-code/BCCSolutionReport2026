# BCC — 2026 Website Usability Report

A scroll-snap presentation reporting the before-and-after results of the BCC Bangkok Cable 2026 website redesign.

## Usage

Open `index.html` in any modern browser. No build step required.

> **Note:** Requires an internet connection to load fonts from Google Fonts.

## Files

- `index.html` — Main presentation (self-contained, all media embedded)
- `bcc-logo.svg` — BCC Bangkok Cable logo
